# oystercardproblem
Required to model fare card system which is a limited version of London’s Oyster card system.

## Build & Run
Build with maven (mvn clean package). Then run with Java: java -jar oystercardproblem-0.0.1-SNAPSHOT.jar

### Demo
Follow command line intruction to demo following flow 

- Loading a card with £30, and taking the following trips, and then viewing the balance.
- Trip 1. Tube Holborn to Earl’s Court
- Trip 2. 328 bus from Earl’s Court to Chelsea
- Trip 3. Tube Earl’s court to Hammersmith
